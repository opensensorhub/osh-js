Class: LeafletView

LeafletView(propertiesopt)

This class is in charge of displaying GPS/orientation data by adding a marker to the Leaflet Map object.

Constructor

new LeafletView(propertiesopt)

Create a View.
Parameters:
Name Type Attributes Default Description
properties Object <optional>
 {} the properties of the view
Properties
Name Type Attributes Default Description
container String The div element to attach to
layers Array.<Object> <optional>
 [] The initial layers to add
autoZoomOnFirstMarker Boolean <optional>
 false auto zoom on the first added marker
follow Boolean <optional>
 false follow the marker
initialView Object <optional>
 Sets the view of the map (geographical center and zoom) with the given animation options. See details
overlayLayers Array.<Object> <optional>
 L.tileLayer objects to use as overlay layer
baseLayers Array.<Object> <optional>
 L.tileLayer objects to use as base layer
Source:
  • core/ui/view/map/LeafletView.js, line 49
Example
import LeafletView from 'core/ui/view/map/LeafletView.js';

 let leafletMapView = new LeafletView({
	container: 'leafletMap',
	layers: [
	  new PointMarkerLayer({
      dataSourceId: gpsDataSource.id,
      getLocation: (rec) => ({
        x: rec.location.lon,
        y: rec.location.lat,
        z: rec.location.alt
      }),
      icon: './images/car-location.png',
      iconSize: [32, 64],
      iconAnchor: [16, 65],
      name: 'Car',
      description: 'GPS car Toulouse'
    })
  ],
	autoZoomOnFirstMarker: true
});
